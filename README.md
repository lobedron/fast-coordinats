# 🌌 Fast Coordinats

**Fast Coordinats** — это удобный веб-инструмент для быстрого пересчета координат между Обычным миром (Overworld) и Адом (Nether) в Minecraft. 

## 🚀 Основные возможности

* **Мгновенный пересчет:** Автоматическое преобразование координат в соотношении 1:8 и обратно.
* **Гибкое управление:** Легкое переключение миров одной кнопкой.

## 🛠 Технологии

Проект построен на чистом стеке без использования тяжелых фреймворков:
* `HTML5`
* `CSS3` (с использованием Flexbox и CSS Variables)
* `JavaScript` (ES6+)

## 📦 Как использовать

1. Перейдите на сайт проекта: [ссылка](https://lobedron.github.io/fast-coordinats/)
2. Скопируйте координаты из игры (например, `X: 100, Y: 64, Z: 200`).
3. Нажмите кнопку **Вставить** (📋) в соответствующей панели.
4. Получите готовые координаты для противоположного мира.
5. Нажмите кнопку **Копировать** (📄) для быстрого переноса в игру.
